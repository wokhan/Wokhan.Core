# UnitFormatter.FormatValue method

Formats a given value (size, bandwitdth, ...) using the default units (["", "K", "M", "G", "T"]) or the specified one. For instance FormatValue(1_000_000, "bps") will return "1Mbps" and FormatValue(2_345_000, units: ["", "KiB", "MiB", "GiB", "TiB"]) will return "2.345MiB".
